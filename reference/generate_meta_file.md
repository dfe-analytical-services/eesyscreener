# Generate a meta file for testing

This function is designed to be used within `generate_test_dfs` to
complement the data file generated by `generate_data_file`.

## Usage

``` r
generate_meta_file(num_filters, num_indicators)
```

## Arguments

- num_filters:

  The number of filters to generate

- num_indicators:

  The number of indicators to generate

## Value

A data frame following the EES metadata standards
