# Local enterprise partnerships lookup

A data frame of valid local enterprise partnership code and name
combinations as used in EES data files.

## Usage

``` r
acceptable_leps
```

## Format

A data.frame with columns:

- local_enterprise_partnership_code:

  GSS LEP code

- local_enterprise_partnership_name:

  Local enterprise partnership name matching the code

## Source

dfe-published-data-qa screener app repo, data/leps.csv
