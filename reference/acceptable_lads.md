# Local authority districts lookup

A data frame of valid local authority district code and name
combinations as used in EES data files.

## Usage

``` r
acceptable_lads
```

## Format

A data.frame with columns:

- lad_code:

  GSS LAD code

- lad_name:

  Local authority district name matching the code
