# Example filter group metadata

This data set contains example metadata for the example filter group
data and does not include a row for the filter grouping column.

## Usage

``` r
example_filter_group_meta
```

## Format

A data frame with 3 rows and 8 columns:

- col_name:

  Name of column in data file

- col_type:

  Type of column in data file

- label:

  Human readable label for column

- indicator_grouping:

  Grouping for indicator columns

- indicator_unit:

  Unit prefix or suffix

- indicator_dp:

  Number of decimal places

- filter_hint:

  Optional hint text to accompany filter

- filter_grouping_column:

  Grouping column showing a filter hierarchy
