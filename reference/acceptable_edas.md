# English devolved areas lookup

A data frame of valid English devolved area code and name combinations
as used in EES data files.

## Usage

``` r
acceptable_edas
```

## Format

A data.frame with columns:

- english_devolved_area_code:

  GSS English devolved area code

- english_devolved_area_name:

  English devolved area name matching the code

## Source

dfe-published-data-qa screener app repo, data/english_devolved_areas.csv
