# Regex pattern for potential observational unit columns

A regex pattern that matches common geography-related and other
observational unit column names (e.g. names containing prefixes like
"la", "reg", "pcon" or suffixes like "name", "code", "urn", "ukprn").

## Usage

``` r
potential_ob_units_regex
```

## Format

A length 1 character vector containing a regex pattern
