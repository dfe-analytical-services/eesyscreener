# Data dictionary

Data dictionary

## Usage

``` r
data_dictionary
```

## Format

A data.frame with columns:

- col_name:

  Standard column names

- col_type:

  Column types of defined columns: "Filter" or "Indicator"

- filter_item:

  Possible filter items within standard filters

- col_name_parent:

  Standardised parent (filter grouping) column names

- filter_item_parent:

  Possible parent (filter grouping) filter items

- cbds_code:

  Common basic data set code where applicable

- item_error_flag:

  Flag as to whether this applies to API data sets or all data sets

- contexts:

  Context or publication area of definition
