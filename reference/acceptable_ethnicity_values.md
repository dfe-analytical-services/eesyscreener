# Acceptable ethnicity values

A data frame of valid GSS ethnicity major and minor value pairs as used
in EES data files.

## Usage

``` r
acceptable_ethnicity_values
```

## Format

A data.frame with columns:

- ethnicity_major:

  GSS ethnicity major group

- ethnicity_minor:

  GSS ethnicity minor group

## Source

dfe-published-data-qa screener app repo, data/ethnicity.csv
