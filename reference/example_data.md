# Example data

Simple example data file.

## Usage

``` r
example_data
```

## Format

A data frame with 9 rows and 8 columns:

- time_period:

  Time period of row

- time_identifier:

  Time identifier of row

- geographic_level:

  Geographic level of row

- country_code:

  Country code (9 digits)

- country_name:

  Country name

- sex:

  Sex of pupil

- education_phase:

  Phase of education

- enrolment_count:

  Number of enrolments

## Source

maintained by explore.statistics@education.gov.uk
