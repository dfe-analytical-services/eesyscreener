# Example filter group with extra row data

Simple example data file showing an example of filter groups in action

## Usage

``` r
example_filter_group_wrow
```

## Format

A data frame with 9 rows and 8 columns:

- time_period:

  Time period of row

- time_identifier:

  Time identifier of row

- geographic_level:

  Geographic level of row

- country_code:

  Country code (9 digits)

- country_name:

  Country name

- establishment_type_group:

  Type of establishment

- education_phase:

  Education phase

- enrolment_count:

  Number of enrolments
